Identify void elements and how they are denoted

A void element is an element whose content model never allows it to have contents under any circumstances. Void elements can have attributes.

Identify attributes

In HTML, an attribute is a characteristic of a page element, such as font size or color.

Write down the correct DOCTYPE declaration for html 5

The declaration must be the very first thing in your HTML document, before the html tag.

Write down syntactically correct statements for the following elements

Article (article)

The article tag specifies independent, self-contained content. An article should make sense on its own and it should be possible to distribute it independently from the rest of the site.

Body (body)

The element contains all the contents of an HTML document, such as text, hyperlinks, images, tables, lists, etc.

line break (br/)

The br tag inserts a single line break.

Headings (h1, ...)

The h1 to h6 tags are used to define HTML headings.

Section with meta-information (head)

The element is a container for all the head elements. The element can include a title for the document, scripts, styles, meta information, and more.

Top level element (html)

The tag tells the browser that this is an HTML document.

Image named Team.jpg (img src="Team.jpg")

The tag defines an image in an HTML page. The tag has two required attributes: src and alt.

Paragraph (p)

The

tag defines a paragraph.

Section (section)

The section tag defines sections in a document, such as chapters, headers, footers, or any other sections of the document.

Label appearing in the title bar (title)

The title tag is required in all HTML documents and it defines the title of the document.

Link to other pages (a)

The tag defines a hyperlink, which is used to link from one page to another.

Comment (!--...--)

The comment tag is used to insert comments in the source code. Comments are not displayed in the browsers.
